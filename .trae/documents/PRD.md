# 星知大陆——小学一至三年级语数英补习课程平台 PRD

## 1. 产品概述

面向中国成都地区小学一至三年级成绩薄弱女孩的补习课程Web平台，融合全球最先进教学法（新加坡CPA、自然拼读、RTI分层干预等），以游戏化冒险世界"星知大陆"为载体，支持家长陪伴学习模式，提供完整的学-练-复教学闭环。

* 目标用户：6-9岁女孩（学习者）+ 家长（陪伴者）

* 核心价值：让成绩差的孩子从"害怕学习"变为"期待学习"，系统化补齐一至三年级语数英知识

## 2. 核心功能

### 2.1 用户角色

| 角色      | 进入方式 | 核心权限                  |
| ------- | ---- | --------------------- |
| 学习者（孩子） | 直接进入 | 浏览课程、完成课时、玩游戏、查看奖励    |
| 陪伴者（家长） | 切换模式 | 查看教学指引、控制进度、评价表现、查看报告 |

### 2.2 功能模块

1. **首页**：学习冒险地图、今日任务、签到、快捷入口
2. **课程浏览页**：学科选择、年级选择、单元列表、课时列表
3. **课时学习页**：核心学习页面，含动画教学、互动练习、游戏化挑战
4. **家长仪表盘**：教学话术、进度控制、评价按钮、学习报告
5. **奖励中心**：星星/钻石余额、徽章展示、心愿商店、成就墙
6. **复习中心**：间隔复习队列、闪卡练习、错题本

## 4. 用户界面设计

### 4.1 设计风格

* **主题**：温暖童趣冒险风——圆润的形状、柔和的渐变、可爱的角色

* **主色**：暖橙色 #FF8C42（活力、温暖）+ 天蓝色 #4ECDC4（清新、信任）

* **辅色**：语文区薰衣草紫 #9B59B6、数学区翡翠绿 #2ECC71、英语区珊瑚红 #E74C3C

* **按钮风格**：大圆角（rounded-2xl）、微3D阴影、按压反馈动画

* **字体**：标题用 ZCOOL KuaiLe（站酷快乐体，童趣感），正文用 Noto Sans SC

* **布局**：卡片式布局、大间距、大按钮（适合触屏）

* **图标/表情**：圆润线条图标 + 可爱表情角色

### 4.2 页面设计概览

| 页面名称  | 模块名称   | UI元素                                       |
| ----- | ------ | ------------------------------------------ |
| 首页    | 学习冒险地图 | 手绘风格地图背景、三大区域用不同颜色标注、路径连线、角色在当前位置、GSAP路径动画 |
| 首页    | 今日任务   | 圆角卡片列表、每科一个任务卡、进度条、星星图标                    |
| 课程浏览页 | 学科选择   | 三张大卡片、各学科主题色、角色形象、进度百分比                    |
| 课时学习页 | 教学动画区  | 全宽动画区域、GSAP CPA过渡、角色精灵动画                   |
| 课时学习页 | 练习区    | 大号选项按钮、拖拽区域、书写板、即时反馈动画                     |
| 课时学习页 | 家长面板   | 侧边栏抽屉、教学话术卡片、表情评价按钮（😊🤔😟）                |
| 奖励中心  | 徽章展示   | 圆形徽章网格、金色/银色/铜色光泽、GSAP弹出动画                 |

### 4.3 响应式设计

* 桌面优先（1920×1080为基准）

* 平板适配（768×1024）：家长面板从侧边栏变为底部抽屉

* 手机适配（375×667）：简化布局，家长面板为浮动按钮触发

### 4.4 动画设计

* 页面切换：GSAP场景过渡（滑入/淡入）

* 课程开场：角色出场动画+标题弹性落入

* CPA过渡：实物→图示→符号的三阶段渐变

* 答题反馈：对勾弹出/叉号抖动+角色表情变化

* 奖励动画：星星弧线飞入收集瓶、徽章旋转弹出

* 闯关成功：彩色纸屑飘落+庆祝文字
