# 老田和小甜甜的游戏屋——小学一至三年级语数英亲子共学补救课程平台 PRD

## 1. 产品概述

**老田和小甜甜的游戏屋** 是一套面向小学 1-3 年级（语文、数学、英语三科）的家庭亲子共学补救学习系统，融合全球最先进教学法（新加坡 CPA、自然拼读、RTI 分层干预、显性直接教学等），以游戏化冒险世界为载体，强制家长陪伴学习，提供完整的"诊断→示范→合练→自练"教学闭环。

> 项目名固定为"**老田和小甜甜的游戏屋**"（老田为父亲，小甜甜为女儿），源自一位父亲用 AI 弥补陪伴缺位的真实故事，任何场合不得写成"老甜甜和小甜甜的游戏屋"。

* 目标用户：6-9 岁女孩（学习者）+ 家长（陪伴者），尤其针对孩子学习出现滑坡、家长想辅导却不知从何下手的家庭
* 核心价值：让成绩差的孩子从"害怕学习"变为"期待学习"，系统化补齐一至三年级语数英知识，同时重建亲子陪伴关系
* 核心场景：家长下班后陪孩子用 30 分钟完成一节"I Do（家长示范）→ We Do（亲子合练）→ You Do（孩子自练）"的完整补救课时

## 2. 核心功能

### 2.1 用户角色

| 角色      | 进入方式 | 核心权限                  |
| ------- | ---- | --------------------- |
| 学习者（孩子） | 直接进入 | 浏览课程、完成课时、玩游戏、查看奖励    |
| 陪伴者（家长） | 切换模式 | 查看教学指引、控制进度、评价表现、查看报告、PIN 管理商品与库存 |

### 2.2 功能模块

1. **首页**：学习冒险地图、今日任务、签到、快捷入口
2. **课程浏览页**：学科选择、年级选择、单元列表、课时列表
3. **课时学习页**：核心学习页面，含智能动画教学、互动练习、游戏化挑战、网课视频匹配
4. **家长仪表盘**：教学话术、进度控制、评价按钮、学习报告、商品库存管理
5. **奖励中心**：星星/钻石余额、徽章展示、心愿商店（库存）、钻石商店（库存）、成就墙
6. **复习中心**：间隔复习队列、闪卡练习、错题本

## 4. 用户界面设计

### 4.1 设计风格

* **主题**：温暖童趣冒险风——圆润的形状、柔和的渐变、可爱的角色

* **主色**：暖橙色 #FF8C42（活力、温暖）+ 天蓝色 #4ECDC4（清新、信任）

* **辅色**：语文区薰衣草紫 #9B59B6、数学区翡翠绿 #2ECC71、英语区珊瑚红 #E74C3C

* **按钮风格**：大圆角（rounded-2xl）、微3D阴影、按压反馈动画

* **字体**：标题用 ZCOOL KuaiLe（站酷快乐体，童趣感），正文用 Noto Sans SC

* **布局**：卡片式布局、大间距、大按钮（适合触屏）

* **图标/表情**：圆润线条图标 + 可爱表情角色

### 4.2 页面设计概览

| 页面名称  | 模块名称   | UI元素                                       |
| ----- | ------ | ------------------------------------------ |
| 首页    | 学习冒险地图 | 手绘风格地图背景、三大区域用不同颜色标注、路径连线、角色在当前位置、GSAP路径动画 |
| 首页    | 今日任务   | 圆角卡片列表、每科一个任务卡、进度条、星星图标                    |
| 课程浏览页 | 学科选择   | 三张大卡片、各学科主题色、角色形象、进度百分比                    |
| 课时学习页 | 智能教学动画区  | 全宽动画区域、智能推断动画场景、角色精灵动画、无纸箱占位符              |

* 桌面优先（1920×1080为基准）

* 平板适配（768×1024）：家长面板从侧边栏变为底部抽屉

* 手机适配（375×667）：简化布局，家长面板为浮动按钮触发

### 4.4 动画设计

* 页面切换：GSAP场景过渡（滑入/淡入）

* 课程开场：角色出场动画+标题弹性落入

* CPA过渡：实物→图示→符号的三阶段渐变

* 答题反馈：对勾弹出/叉号抖动+角色表情变化

* 奖励动画：星星弧线飞入收集瓶、徽章旋转弹出

* 闯关成功：彩色纸屑飘落+庆祝文字

## 5. 竞争优势与差异化

### 5.1 与同类产品的核心差异

| 维度 | 同类教育 App | 老田和小甜甜的游戏屋 |
|------|------------|---------------------|
| **网课视频** | 硬编码直链，易失效、常匹配错课 | 动态搜索链接生成，每课时 3 个免费权威源，永不失效 |
| **教学动画** | 通用占位符或固定模板 | 智能动画推断系统，根据内容自动选择 14 种场景，无纸箱占位符 |
| **奖励机制** | 无限发放，缺乏稀缺感 | 库存经济系统，售罄置灰，培养延迟满足 |
| **家长角色** | 扔给孩子自己学 | 强制亲子共学，I Do→We Do→You Do |
| **运行方式** | 必须联网 | 完全离线，单文件双击即用 |
| **部署** | 单平台 | 一套代码 Web/Windows/Android 三端 |

### 5.2 商业价值

| 维度 | 说明 |
|------|------|
| **内容壁垒** | 9 个课程包、72 个单元、360 课时、2520 道练习题，三科完全对等且对标最新课标 |

| **儿童心理学** | 双货币激励、库存稀缺感、成就解锁、个性化表扬，正向行为强化 |

### 5.3 儿童心理学考量

游戏屋在设计中深度融入了儿童心理学原则：

- **奖励系统**：星星（日常）+ 钻石（成就）双货币体系，配合**库存稀缺感**（心愿商店库存有限、售罄置灰），让孩子学会"攒星星—做选择—延迟满足"
- **学习体验**：I Do → We Do → You Do 三段式流程提供安全感支架，正确率 ≥80% 才过关避免挫败感
- **个性化反馈**：可设置孩子名字，答对题时 TTS 朗读"小甜你真棒"，让孩子感受到被看见
- **多感官刺激**：视觉（智能动画+emoji）+ 听觉（TTS 语音）+ 动觉（拖拽题目），符合蒙台梭利多感官学习法
- **情感连接**：项目本身是"父亲写给女儿的道歉信"，产品调性温暖而非冷冰冰

## 6. 非功能性需求

* **离线运行**：单文件打包（vite-plugin-singlefile），构建产物为单个 HTML 文件，支持 `file://` 协议直接打开，无需联网
* **哈希路由**：`file://` 协议下自动启用 createWebHashHistory，避免刷新 404
* **本地存储**：所有进度、奖励、交易历史、设置持久化到 localStorage，隐私安全
* **跨平台**：Web（浏览器）/ Electron（Windows 桌面便携 EXE + NSIS 安装包）/ Capacitor（Android APK）
* **性能**：单课时 30 分钟，I Do 仅 6 分钟，You Do 正确率 ≥80% 才过关
* **语音**：基于 Web Speech API 的 TTS，使用系统自带语音，完全离线，含拼音智能转换与 emoji 清洗
